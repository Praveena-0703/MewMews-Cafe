# MewMew's Cozy Cafe

MewMew's Cozy Café is a cozy cat café simulation game built entirely in **Scratch**. Players welcome adorable feline customers, listen carefully to their orders, and serve the correct food while enjoying hand-drawn artwork, custom animations, sound effects, background music, and randomized gameplay.

![Cover](assets/cover.png)

## Gameplay:

![Gameplay](assets/gameplay.gif)

## Features:

- Four unique cat customers with individual personalities
- Randomized food orders for every playthrough
- Heart-based scoring system
- Background music and sound effects
- Hand-drawn sprites, backgrounds, and menu items
- Character animations and interactive dialogue
- Game Over screen after serving five customers

## How to Play:

1. Click the **Green Flag** to open the café.
2. Welcome each customer and listen carefully to their order.
3. Select the correct food item from the menu.
4. Earn a heart for every correct order.
5. Serve five customers to complete the day.

## Built With:

- Scratch 3.0

## About the Project:

Every sprite, background, and menu item in this project was hand-drawn by me. I created MewMew's Cozy Café to practice event-driven programming, variables, broadcasts, animations, and user interaction while designing a fun and polished gameplay experience.
